# 同题多目标：页面导演回归样例

主题：一个面向职场新人的周报复盘工具。

| 主目标 | 标题承诺 | 封面版式 | 文字密度 | 内页角色组合 | 产品出现方式 |
|---|---|---|---|---|---|
| 收藏 | “周报写不出来时，先做这 3 步” | E 信息主导卡 | 中高；步骤和清单为主 | 结果预览 -> 步骤 -> 易错点 -> 清单 | 方法载体，半显性 |
| 信任 | “我以前每周五都怕写周报，后来只改了一个顺序” | B 上文下景 | 低；场景和过程证据为主 | 代入 -> 过程 -> 证据 -> 限制 -> 收束 | 背景工具，隐性 |
| 咨询 | “周报总是写成流水账的人，先看这 4 个判断标准” | A 左文右物/人 | 中；判断标准与边界为主 | 人群分型 -> 标准 -> 对比 -> 适用/不适用 -> 行动 | 直接方案，显性但克制 |

通过：三条内容的标题、封面构图、文字密度、页面角色和产品露出至少三项不同；失败：只换几个标题词，其余页面照抄。
