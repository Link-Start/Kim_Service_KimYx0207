# 创作示例

## 可见输出

先围绕一句能被记住的话做产物，不要围绕完整体系做产物。如果接收者看完第一屏都复述不出重点，更多章节只会让失败变重。

第一版应该像这样：一个锋利判断、一个具体例子、一个立刻能做的动作。接收者看完要能想明白：“我知道这是干什么的，也知道下一步怎么用。”

## 方案

先做 200 字短版，再扩完整产物。

- 输入：一个主张、一个例子、一个目标接收者。
- 动作：写出最短版本，让接收者从看见走到行动。
- 输出：标题、证明/例子、一个下一步动作。
- 通过标准：接收者能复述主张，并说出下一步。
- 停止标准：如果对方只说“挺有意思”，但说不出改变什么，核心承诺是虚的。

## 后台检查

- 本质：只有一个控制性想法。
- 叙事：从后果或冲突出发。
- 证据：一个具体例子比大段解释更能建立信任。
